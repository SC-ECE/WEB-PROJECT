
# Food website
It is a complete responsive food website, built using HTML,CSS and javascript(JS).


## Authors

- [@dipanwitb001](https://github.com/dipanwitb001)


## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
